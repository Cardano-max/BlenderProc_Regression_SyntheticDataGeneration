Place your HDRI environment maps here for realistic lighting.

## Recommended Sources (Free):
- [Poly Haven](https://polyhaven.com/hdris) - High quality, CC0 license
- [HDRI Haven](https://hdrihaven.com/) - Merged with Poly Haven
- [Pixar Free HDRIs](https://renderman.pixar.com/view/free-hdri)

## Suggested HDRIs for Outdoor Industrial Scenes:
- `alps_field.hdr` - Open field with mountains
- `industrial_sunset.hdr` - Factory/industrial setting
- `overcast_day.hdr` - Soft diffused lighting
- `sunny_field.hdr` - Bright daylight

## Supported Formats:
- `.hdr` (Radiance HDR - recommended)
- `.exr` (OpenEXR - high dynamic range)

## Resolution:
- 2K (2048x1024) - Good for most uses
- 4K (4096x2048) - Higher quality reflections
- 8K+ - Only if needed for close-up reflections

## Download Example:
```bash
# Using wget to download from Poly Haven
wget https://dl.polyhaven.org/file/ph-assets/HDRIs/hdr/2k/alps_field_2k.hdr
```
